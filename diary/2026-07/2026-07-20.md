### Git 常用命令

```bash
# 克隆仓库
git clone git@github.com:用户名/仓库名.git

# 查看状态
git status

# 添加文件
git add 文件名    # 或 git add . 添加全部

# 提交
git commit -m "描述信息"

# 推送到 GitHub
git push

# 拉取最新
git pull
```
