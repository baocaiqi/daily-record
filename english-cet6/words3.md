### ⚠️ 形近词 / 近义词辨析

| 词对 | 区别 |
|------|------|
| **paradigm** vs **parameter** | paradigm = 范式/模式（理论框架）；parameter = 参数（可调节变量） |
| **fur** vs **leather** | fur = 动物皮毛（带毛）；leather = 皮革（去毛鞣制后） |
| **gaze** vs **stare** vs **glance** | gaze = 凝视（含情感，时间长）；stare = 盯着（可能不礼貌）；glance = 一瞥（快速扫过） |
| **preface** vs **foreword** vs **introduction** | preface = 作者自序；foreword = 他人作序/推荐序；introduction = 导论/引言 |
| **fury** vs **anger** vs **rage** | fury = 暴怒（最强烈，近乎失控）；rage = 狂怒；anger = 生气（最通用） |
| **henceforth** vs **from now on** | henceforth = 正式/法律用语；from now on = 日常口语 |
| **albeit** vs **although** | albeit = 正式，多接形容词/短语；although = 后接完整从句 |
