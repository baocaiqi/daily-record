### ⚠️ 形近词 / 近义词辨析

| 词对 | 区别 |
|------|------|
| **communism** vs **communist** | communism = 共产主义（制度/理念）；communist = 共产主义的 / 共产主义者 |
| **inhabitant** vs **habitat** | inhabitant = 居民，居住者；habitat = （动植物的）栖息地 |
| **symphony** vs **concerto** | symphony = 交响乐；concerto = 协奏曲 |
| **qualitative** vs **quantitative** | qualitative = 定性的；quantitative = 定量的，量性的 |
| **discreet** vs **discrete** | discreet = 谨慎的；discrete = 离散的，分开的 |
| **barber** vs **hairdresser** | barber = 传统男理发师；hairdresser = 通用美发师 |
| **centigrade** vs **Celsius** | 均指摄氏度，centigrade 较老式，Celsius 更常用 |
| **fantastical** vs **fantastic** | 同义，fantastical 更偏文学/古雅 |
| **burglar** vs **thief** vs **robber** | burglar = 入室窃贼；thief = 小偷（最通用）；robber = 强盗（多含暴力/威胁） |
