## 📖 使用说明

每天创建一个新的日记文件，格式：`diary/YYYY-MM/YYYY-MM-DD.md`

```bash
# 创建今天的日记
touch diary/2026-07/2026-07-20.md

# 写完提交
git add .
git commit -m "2026-07-20 学习记录"
git push
```
